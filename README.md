# Enterprise Admin Dashboard 

A modern, robust, and highly scalable Admin Dashboard designed to streamline financial data tracking and warehouse stock management. Built using **React.js** and **TypeScript**, this application prioritizes peak performance, type safety, and an intuitive user experience.

---

## 🚀 Tech Stack

This project leverages a modern frontend ecosystem to ensure clean code architecture and seamless maintainability:

* **Core Framework:** React.js (v18+)
* **Language:** TypeScript (for strict type safety and robust autocompletion)
* **State Management:** Redux Toolkit (efficient global state management for finance and inventory data)
* **UI Library:** Material UI (MUI) v5 (for consistent, responsive components based on Google's Material Design)
* **Form & Validation:** Formik + Yup (handling complex form states and powerful schema validation)
* **Component Playground:** Storybook (for isolated component development and documentation)

---

## ✨ Key Features

### 1. Financial Data Management
* **Transaction Ledger:** Full CRUD (Create, Read, Update, Delete) functionality to record and monitor daily financial transactions.
* **Financial Reporting:** Advanced data filtering by date range, category, or status for auditing purposes.

### 2. Warehouse Stock Management
* **Product Management:** Organize products by categories, vendors/suppliers, and specific warehouse aisle/shelf locations.
* **Inventory Audit Logs:** Detailed mutation logs to track every single movement of stock items.
